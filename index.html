Questions

Questions

What counts as one page?

Each PDF page processed counts as one credit.

Do credits expire?

No. Credits never expire. Buy when you need them, use them when you're ready.

What GIS formats can I export?

Esri File Geodatabase (.gdb), Shapefile (.shp), GeoJSON, and KML.

Do I need ArcGIS to use DrawBridge?

No. DrawBridge runs in your browser. The exported formats are compatible with ArcGIS Pro, ArcGIS Online, QGIS, and any GIS software that reads standard formats.

Is my data stored permanently?

Session data is temporary. Sessions expire after 24 hours. Export and save your .gdb before closing.

Does it work on tablet or mobile?

DrawBridge is designed for desktop use. A large screen makes digitizing much easier.
